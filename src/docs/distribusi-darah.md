# Modul Distribusi Darah

## Tujuan

Modul Distribusi Darah digunakan untuk mencatat proses penyerahan kantong
darah dari Unit Donor Darah kepada Rumah Sakit.

## Alur

```text
Permintaan darah disetujui
→ Kantong darah dialokasikan
→ Seluruh kebutuhan terpenuhi
→ Permintaan berstatus siap diambil
→ Distribusi dijadwalkan
→ Distribusi ditandai siap
→ Darah diserahkan
→ Distribusi selesai
