# Modul Alokasi Darah

## Tujuan

Modul Alokasi Darah digunakan untuk menghubungkan kantong darah yang
tersedia dengan permintaan darah Rumah Sakit.

## Alur

```text
Rumah Sakit mengajukan permintaan
→ Admin atau Petugas menyetujui permintaan
→ Sistem mencari stok sesuai
→ Kantong darah dikunci
→ Kantong dialokasikan
→ Permintaan siap diambil jika kebutuhan terpenuhi
