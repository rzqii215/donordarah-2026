# Modul Pendaftaran Donor

## Tujuan

Modul Pendaftaran Donor menghubungkan Pendonor dengan Jadwal Donor.

## Alur

```text
Pendonor memilih jadwal
→ Mengisi skrining awal
→ Pendaftaran berstatus menunggu
→ Admin atau Petugas meninjau
→ Disetujui atau ditolak
→ Pendonor datang ke lokasi
→ Petugas mencatat kehadiran
