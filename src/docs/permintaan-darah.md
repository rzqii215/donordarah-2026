# Modul Permintaan Darah

## Tujuan

Modul Permintaan Darah digunakan Rumah Sakit untuk mengajukan kebutuhan
darah kepada Unit Donor Darah.

## Alur

```text
Rumah Sakit terverifikasi
→ Membuat permintaan darah
→ Permintaan diajukan
→ Admin atau Petugas meninjau
→ Disetujui, ditolak, atau menunggu stok
→ Kantong darah dialokasikan
→ Darah siap diambil
→ Distribusi selesai
