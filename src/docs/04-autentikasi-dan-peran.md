# Autentikasi dan Peran Pengguna

## 1. Tujuan

Dokumen ini menjelaskan struktur pengguna, role, status akun, dan hak akses
pada Sistem Informasi Manajemen Unit Donor Darah.

Sistem menggunakan:

- Laravel Authentication
- Spatie Laravel Permission
- Filament v3
- Livewire dan Blade
