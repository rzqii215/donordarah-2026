# Modul Pemeriksaan Kesehatan

## Tujuan

Modul Pemeriksaan Kesehatan digunakan Petugas untuk mencatat hasil
pemeriksaan Pendonor sebelum proses donor darah dilakukan.

## Alur

```text
Pendaftaran disetujui
→ Pendonor datang
→ Petugas mencatat kehadiran
→ Petugas melakukan pemeriksaan
→ Layak atau Tidak Layak Donor
