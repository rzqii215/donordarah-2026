# Modul Kantong Darah

## Tujuan

Modul Kantong Darah digunakan untuk mencatat darah yang berhasil diambil
dari Pendonor dan mengelola status stoknya.

## Alur

```text
Pendonor hadir
→ Pemeriksaan kesehatan
→ Pendonor dinyatakan layak
→ Proses donor dilakukan
→ Kantong darah dibuat
→ Pemeriksaan mutu
→ Tersedia atau ditolak
